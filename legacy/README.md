# Legacy assets

These files are kept for historical reference only. They were part of the original
Travlr Getaways static prototype before it was converted to the MEAN stack.

Nothing in the running application depends on this folder:

- The seven HTML files are early static mockups. The live customer site uses
  Handlebars templates in `app_server/views/`.
- `server.js` was an early Express bootstrap that only served static files.
  The real entry point is `app.js` in the project root.

Safe to delete if you do not need the history.
