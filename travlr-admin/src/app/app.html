Getting around

  • Dashboard shows live revenue, bookings, and category mix from the database.
  • Trip Catalog lists every package on the customer site. Search, sort, and export to CSV.
  • New Trip opens a form to add a package. Validation runs against the same schema as the API.
  • Customers is the registered-account list. Admin-only.

Default admin login
